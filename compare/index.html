ARXplore v1.0

ARXplore v1.0 is a web tool designed to help users view and understand Automotive Open System Architecture (ARXML) or standard XML files.
Users can upload one or two files, which are then parsed and displayed as interactive hierarchical tree views.
When two files are provided, the tool performs a side-by-side comparison, visually highlighting additions, deletions, and modifications.
Additionally, it features an optional AI-powered analysis to generate a natural language summary of the detected differences.
